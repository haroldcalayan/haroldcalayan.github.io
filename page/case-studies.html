The Challenge

Philippine Airlines needed a robust mobile solution to handle complex booking flows, real-time flight tracking, and loyalty program integration. The existing system faced performance bottlenecks and lacked a modern UI/UX.

The Solution

I implemented Clean Architecture with MVVM to ensure separation of concerns and testability. Leveraging Jetpack Compose, I crafted a fluid, responsive interface that adheres to brand guidelines. Data fetching was optimized using Coroutines and Flows, reducing load times by 40%.
