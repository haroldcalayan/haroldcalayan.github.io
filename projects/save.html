Project Overview

This project is an entry for a national competition within the company. It is a utility mobile app that showcases user's current and forecasted electric usage, tips recommendation through Artificial Intelligence (AI), and the current usage of each appliance with the help of Augmented Reality (AR).

Driving financial inclusion through an intuitive and secure mobile banking experience with automated savings rules.

Current & forecasted electric usage
